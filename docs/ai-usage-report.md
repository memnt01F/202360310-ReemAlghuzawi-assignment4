# AI Usage Report – Assignment 4

## Tools Used & Use Cases

Artificial Intelligence tools were used throughout the development of this project to support coding, debugging, and documentation.

Primary tool used:

ChatGPT

Use cases:

- Debugging JavaScript errors
- Implementing animations
- API integration support
- Writing documentation
- Explaining programming concepts

AI was used as a learning assistant and development support tool.

---

## Benefits

Using AI provided several advantages:

- Faster development process
- Clear explanations of programming concepts
- Improved debugging efficiency
- Better code organization
- Increased productivity
- Improved understanding of JavaScript logic

---

## Challenges

Some challenges were experienced:

- AI suggestions required testing
- Generated code sometimes needed adjustments
- Understanding the logic was necessary
- Manual verification was required

These challenges helped strengthen problem-solving skills.

---

## Learning Outcomes

Using AI helped develop important technical skills.

Key learning outcomes:
- DOM manipulation
- Debugging techniques
- Code organization

AI supported learning but the final implementation required personal understanding and testing.

---

## Responsible Use

AI-generated suggestions were reviewed and modified before use.

Responsible practices included:

- Testing all code before implementation
- Editing code to match assignment requirements
- Ensuring originality of work
- Understanding each feature
- Fixing errors manually
- Adding comments and documentation

AI was used responsibly as a support tool rather than a replacement for independent work.
