# Personal Portfolio Web Application – Assignment 4

## Project Overview

This project is the final version of my personal portfolio web application developed throughout the course.  
It combines all previous assignments into a polished, professional, and fully functional website that demonstrates my web development skills.

The application showcases my projects, skills, and technical abilities using modern web technologies. It includes interactive features, dynamic content, and responsive design to create a professional user experience.

---

## Features

### Core Features

- Responsive personal portfolio website
- Skills section with interactive hover animation
- Project search functionality
- Project filtering by category
- Project sorting (A–Z / Z–A)
- GitHub API integration
- Contact form validation
- Scroll progress bar
- Smooth navigation between sections

---

## Innovation Feature

### Interactive Skills Section

The portfolio includes a dynamic skills section where skills are displayed as floating tags.  
When the user hovers over a skill, it animates smoothly to create a modern and interactive user experience.

### Scroll Progress Bar

A progress bar appears at the top of the page and updates as the user scrolls.  
This feature improves usability and demonstrates creative implementation.
